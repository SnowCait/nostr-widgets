# Nostr Widgets

Embed Nostr notes into your website.

## Note

```html
<nostr-note data="{...}"></nostr-note>
```

### Attributes

| attribute | required | value     | example                                      |
| --------- | :------: | --------- | -------------------------------------------- |
| data      |   yes    | Note JSON | `{ "kind": 1, "id": "", "pubkey": "", ... }` |
